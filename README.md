# CookingRecipesApp

Technologies: UIKit(without storyBoards), CleanSwift, API Spoonucilar, SDWebImage

# About

In this project I've tried to make some funchionality of Cooking Recipe's App

 - [x] You can find different type of receipes
 - [x] You can choose receipes and get detailed info :
    - Type of meal
    - Ingridients
    - Mathes for this dish
    - How to cook
